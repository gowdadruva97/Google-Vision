Google Vision Android Application

This project was built during a one month internship at BSQ Technologies , Mysore in July and July 2018. 

The application uses Google Cloud Vision API to capture images, extract text using OCR, translate the detected text into the user's regional language and convert it to speech output.

The app was designed with a specific use case in mind. Travellers visiting foreign countries who cannot read local language signs or notices can point their smartphone camera at any text, and the app will read it aloud in a language they understand.

Key Features :

Real-time image capture and text extraction using OCR and Google Cloud Vision API. Text-to-speech conversion of extracted content for accessibility. Multilingual translation support for travellers and visually impaired users. Google Search integration to surface additional context on detected text. Clean UI built with Android XML layouts including splash screen, options menu and camera capture flow.

Tech Stack :  Java, Kotlin, Android Studio, Google Cloud Vision API, OCR, Android SDK, XML Layouts
